# OMEGA-ULTRA-v4
OMEGA ULTRA v4 to modułowy system multi-agent AI oparty na monorepo. Zawiera orchestrator, routing modeli LLM, pamięć, event bus, kontrolę kosztów oraz rozszerzalne agenty i narzędzia. Umożliwia budowanie skalowalnych workflow AI, automatyzację zadań i współpracę wielu agentów w jednym ekosystemie.
